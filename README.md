# SQL Practice Repository

Welcome to my SQL practice repository. This repo documents my journey of learning and mastering SQL through structured practice, covering concepts, queries, and mistake tracking.

---

## Folder Structure (Per Topic)

Each topic follows a consistent structure:

```
TopicName/
│── TopicName_reference.sql           -- Concepts + tables used
│── TopicName_practice_questions.sql  -- Questions practiced
│── TopicName_mistakes.sql            -- Mistakes + corrections
```

---

## Learning Approach

For every topic, I follow a structured method:

Concept Understanding → Covered in `_reference.sql`
Question Bank → Maintained in `_practice_questions.sql`
Mistake Tracking → Logged in `_mistakes.sql`

This approach helps me:

Strengthen fundamentals,
Improve problem-solving speed,
Avoid repeating mistakes &
Prepare effectively for interviews
